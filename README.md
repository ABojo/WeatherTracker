# WeatherTracker
A weather tracking app built for the <a href='https://www.theodinproject.com/paths/full-stack-javascript/courses/javascript/lessons/weather-app'>Odin Project Curriculum</a>.

You can build a list of locations that you'd like to monitor the current temperature for and have all of the data updated for you upon refresh.

https://abojo.github.io/WeatherTracker/
